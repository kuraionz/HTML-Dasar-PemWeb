APA ITU HTML

HTML (Hypertext Markup Language): Bahasa markup yang digunakan untuk membuat halaman web dan menampilkan informasi di browser. HTML terdiri dari kode tag yang memberikan instruksi kepada browser untuk menampilkan halaman sesuai keinginan pengguna. HTML diatur oleh W3C (World Wide Web Consortium).


SEJARAH HTML

HTML 2.0 disetujui pada 22 September 1995.

HTML 3.2 pada 14 Januari 1996.

HTML 4.0 pada 18 Desember 1997.

XHTML 1.0 pada 24 Januari 2000.


BAGIAN-BAGIAN HTML

Header berisi informasi halaman seperti judul <title>, skrip (JavaScript, VBScript), gaya (CSS), dan meta tag untuk SEO.

Body merupakan bagian yang ditampilkan kepada pengguna, berisi teks, gambar, link, skrip server-side, dan multimedia.
